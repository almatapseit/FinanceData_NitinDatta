# FinanceData_NitinDatta
This project is a data analytics case study exploring investor preferences using Python, SQL, and visualization tools. The analysis is based on an open dataset from Kaggle ("Finance Data (Nitin Datta)") which contains survey responses about investment choices and motivations.
